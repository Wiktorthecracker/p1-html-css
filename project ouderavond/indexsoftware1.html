Software




Welke software gebruiken wij?

Hier op school gebruiken wij Visual Studio Code voor het maken van de websites.


Wat is Visual Studio Code?

Visual Studio Code is een gestroomlijnde code-editor met ondersteuning voor ontwikkelingsbewerkingen zoals foutopsporing, het uitvoeren van taken en versiebeheer.

Het is bedoeld om alleen de tools te bieden die een ontwikkelaar nodig heeft voor een snelle code-build-debug-cyclus en laat complexere workflows over aan vollediger aanbevolen IDE's, zoals Visual Studio IDE.
